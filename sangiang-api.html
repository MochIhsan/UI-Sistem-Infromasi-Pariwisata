Tentang Gunugn Sangiang Api

Gunung Sangiang merupakan gunung berapi aktif yang terletak di Pulau Sangiang, Kecamatan Wera, Kabupaten Bima, Nusa Tenggara Barat. Gunung ini berada di tengah Laut Flores dan dikenal memiliki dua puncak utama, yaitu Doro Api dan Doro Mantoi. Secara geografis, Gunung Sangiang menjadi bagian dari gugusan gunung api aktif di wilayah Indonesia timur dan memiliki peran penting dalam sejarah geologi kawasan Bima.

Selain nilai geologinya, Gunung Sangiang juga memiliki nilai budaya dan sejarah yang kuat bagi masyarakat sekitar. Pulau Sangiang sering dikaitkan dengan cerita rakyat dan legenda lokal yang diwariskan secara turun-temurun. Panorama alam yang memadukan laut, gunung, dan pulau kecil.
